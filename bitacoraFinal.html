Desiciones de diseño

El equipo decidió tener una página principal simple que funcionara como menú de acceso a las páginas individuales de cada integrante.

Se eligió un diseño x para el main con diseños personalizados en la pagina de cada uno

Dificultades encontradas

Al principio las imágenes (avatar y fondos) no se mostraban en dispositivos móviles porque las rutas en el CSS estaban mal.

Se tuvo complicación para que la página fuera responsive en diferentes tamaños de pantalla. Se resolvió aplicando media queries en el CSS.

Cambios importantes durante la implementación

Se pensó tener un estilo "base" al que todos se ajustarían y donde aplicarían sus modificaciones

El estilo base se descartó, dejando que cada miembro usara un estilo libre
